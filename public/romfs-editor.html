ROMFS

ROMFS is a read-only filesystem flashed alongside the firmware and mounted at /rom. It holds AI models, label maps, and other assets, which the firmware reads directly from flash without copying them into RAM.

Buttons

  • Read Device -- pull the image currently flashed on the board.
  • Stock -- load the factory image bundled with the installed firmware.
  • Add Files... -- stage local files (drag & drop also works).
  • Write to Device -- flash the current entries and reset the board.

Tips

  • Right-click a row to mark it for deletion; right-click again to restore.
  • .tflite defaults to 16-byte alignment; other files to 4. Adjust per row if a model needs more.
  • The bar in the status footer tracks used vs. total partition size; writes are blocked when over.
  • Reading or writing the partition can take a long time depending on the partition size and link speed.
  • Press Esc to close.
